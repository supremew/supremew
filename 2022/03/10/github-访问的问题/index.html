github 访问的问题

发表于 2022-03-10 更新于 2022-05-09 分类于 日常生活

github 经常出现访问速度慢， 尤其在下载 github 上的压缩包， 图片等等。 经过多方探访，总算找到了解决办法。
hello the world
通过 ipaddress.com 查询关于 github 的域名的 IP, 然后将所查到的相应 IP 添加到 hosts 中。比较简单的办法就是在 gitee.com 上搜索 github hosts 关键词，找到最新的 hosts 配置， 直接添加到 hosts 中就是了。
