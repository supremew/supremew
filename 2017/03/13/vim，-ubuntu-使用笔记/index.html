2017-03-13-2 vim， ubuntu 使用笔记

发表于 2017-03-13 更新于 2021-12-11 分类于 ubuntu ， vim/gvim

ubuntu 在 8.04 开始就已经将 /dev/sda 和 /dev/hdb 和二唯一了， 现在没有 /dev/hda 这样的 的表达式了。而且建议在以后的分区表达过程中尽量用 uuid 表示。 在 grub2 不管时 ide 还是 scis 等都只有 (hd0）或者什么来表示硬盘。 找到了一个小工具， 可以更改 ubuntu 的默认字体其名字为 Unity Tweaty Tool 。 目前 我将我的字体大多都改成了文泉驿字体， 只有窗口标题的字体还保留为 ubuntu 字体。 vim 今天在而然的机会中发现了中文在指定列后自动插入换行符， 那就是通过在 formatoptions 的后面加上小写的 m。 wordpress 在撰写博客的时候要适时的加入 more 标签。这是为了防止过长的首页博客文章看起来很烦 人。这样也会给来看你博客的人一种厌烦的感觉。 因此在适当的时候加入 more 标签。
