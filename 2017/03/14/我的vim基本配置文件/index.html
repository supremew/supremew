2017-03-14 我的vim基本配置文件

配置文件内容如下 "显示行号 set number "设置文本宽度 set textwidth=80 "设置文本编码 "set fileencodings=utf-8 set encoding=utf-8 set fileencoding=utf-8 set fileencodings=utf-8,default,ucs-bom,latin1 "解决gvim菜单栏出现乱码的问题 "自动对中文文档进行规范化 autocmd BufWritePre *.markdown,*.md,*.text,*.txt,*.wiki,*.cnx call PanGuSpacing() "设定默认工作路径 cd ~/Documents/note "设定不备份文件 "set nobackup ":au BufWritePre * let &bex = '-' . strftime("%Y%b%d%X") . '~' "设定不高亮显示搜索 set nohlsearch "在输入多字节字符时(unicode编码超过255的， 如中文）， 也段行插入换行， 这是我以 "前想找的功能终于找到 "了 "在链接行时， 如果行前， 行后是unicode超过255的字符， 那么就不在中间加入空格。 set formatoptions+=mM set expandtab set softtabstop=8 set shiftwidth=8 "设定开启文件类型检测功能 filetype on "设置标尺 set ruler "设置gvim的字体， 字号 set guifont=WenQuanYi\ Micro\ Hei\ Mono\ 16 "设置语法高亮 syntax on
