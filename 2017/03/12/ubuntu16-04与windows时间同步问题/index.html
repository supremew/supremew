2017-03-13 Ubuntu16.04与Windows时间同步问题

发表于 2017-03-12 更新于 2021-12-11 分类于 ubuntu

刚装完双系统后发现Ubuntu与Windows系统时间总会差8小时，这是因为Ubuntu将BIOS时间认为是世界标准时间，而Windows认为是当地时间。 网上很多解决方法都是修改/etc/default/rcS文件，将UTC=yes改为UTC=no，但是在16.04中，时间由timedatectl管理，修改rcS文件是无效的，现在的方法是： 打开终端，运行命令： timedatectl set-local-rtc 1 –adjust-system-clock timedatectl set-ntp 0 将硬件时钟使用本地时间。 还可以将硬件时钟使用UTC，运行命令： timedatectl set-local-rtc 0 timedatectl set-ntp 1 接着在Windows下修改注册表： Reg add HKLM\SYSTEM\CurrentControlSet\Control\TimeZoneInformation /v RealTimeIsUniversal /t REG_DWORD /d 1
