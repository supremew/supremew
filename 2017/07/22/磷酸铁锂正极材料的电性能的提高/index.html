2017-07-22 磷酸铁锂正极材料的电性能的提高

发表于 2017-07-22 更新于 2021-12-11 分类于 日常生活

简单描述

提高磷酸铁锂正极材料的电性能， 正对目前这种工艺， 有两种措施可以大大的提高其性能。 一、 可以通过某种手段， 除去经砂磨后的物料中的磁性物质（主要是铁）， 磁性物质的存在对其电性能影响是非常大的。 二、 可以在磷酸铁锂初次充电时， 采用恒压充电， 将其电压稳定在3.4V左右或略高于3.4V， 比如3.4~3.5 在接近放电平台附近进行长时间的冲电， 知道电流减少为0， 或充电电量基本无变化为止。

除去磁性物质

可以采用高磁场的电磁铁， 进行除磁， 目前市面上有成熟的除磁设备。 在除去磁性物质时， 不能将磷酸铁锂前驱体浆料的固含量调的太高， 否则会导致除磁性物质时， 除不干净。

恒压充电

先通过恒流充电， 得到充电平台。 然后再尽在略高于充电平台的电压进行充电。
