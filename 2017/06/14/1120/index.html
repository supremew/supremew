发表于 2017-06-14 更新于 2021-12-11 分类于 日常生活

碳酸亚铁母液处理

方案一

  1. 原料 1.1 碳酸亚铁母液

    • ph 6.5 ~ 7.0
    • 铁含量 0.1g/L
    • 体积 1L

    1.2 双氧水

    • 质量分数 20%
    • 体积 10ml

  2. 操作 2.1 在搅拌条件下， 加入 10ml 双氧水； 2.2 搅拌 2h， 静止 2h; 2.3 滤纸过滤后， 取样得到 0614-1-1;

    方案二

  3. 原料 1.1 碳酸亚铁母液

    • ph 6.5 ~ 7.0
    • 铁含量 0.1g/L
    • 体积 1L

    1.2 双氧水

    • 质量分数 20%
    • 体积 10ml

    1.3 氢氧化钠

    • 纯度 工业级
    • 质量 未称量

  4. 操作 2.1 在搅拌条件下， 调节 ph 到 8 ～ 9； 2.2 在搅拌条件下， 加入 10ml 双氧水； 2.3 搅拌 2h， 静止 2h ； 2.4 滤纸过滤， 得到样品 0614-2-1；

方案三

  1. 原料 1.1 样品 0614-1-1
  2. 操作 2.1 将样品 0614-1-1 量取 500ml； 2.2 将其放在 60℃ 的水浴锅中进行保温 2h； 2.3 使用滤纸过滤后， 得到样品 0614-1-2

方案四

  1. 原料 1.1 样品 0614-1-1 1.2 双氧水
    • 含量 20%
  2. 操作 2.1 将样品 0614-1-1 量取 500ml； 2.2 加入 5ml 双氧水， 搅拌 2h， 静止 2h； 2.3 滤纸过滤后得到样品 0614-1-3
