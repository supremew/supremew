vim-oss-image-uploader 的使用

vim-oss-image-uploader 是专门为 vim 开发的一个用于撰写 markdown 文档的图床插件， vim-oss-image-uploader 已经开发了有一段时间了， 看样子是很久没有维护了。而且内部有一部分的 python 代码写得不是很好， 我也对其进行了一些修改。

  • 在这个插件中我增加了一些配置项， 比如图片处理样式;
  • 修复了一些小的bug。 （比如原有的插件中不能使用相对路径， 增加了对相对路径的支持。）

我将修改后的插件代码暂时放在 码云 托管。
