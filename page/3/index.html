2021-12-13 git 的默认编辑器的修改

发表于 2021-12-13 更新于 2021-12-23 分类于 日常生活

git commit

此时是进入GUN nano编辑器。在这里可以添加你的commit imformation 然后ctrl+o,回车保存，再ctrl+x退出。 因为对Nano编辑器很不熟悉，在这里我想将默认编辑器改为vim。打开.git/config文件，在core中添加 editor=vim即可。
或者运行命令 git config –global core.editor vim 修改更加方便。

git commit -a

把所有add了的文件都加入commit，然后进入编辑器编辑commit信息。

git commit -m “commit imformation”

直接在后面添加commit 信息然后提交commit，与gitcommit相比快捷方便，但是就是commit信息格式无法控制。

git commit –amend

修改最后一次commit的信息。

2021-08-27 鱼杆的选择

发表于 2021-08-27 更新于 2021-12-11 分类于 日常生活

目前市面上主要有 37、28、19 调鱼杆(这三种调性的鱼杆均属于硬钓竿， 46 或 55 控鱼性很差基本被淘汰)， 选择什么样的鱼杆主要是看钓鱼场景以及钓鱼让人的经验。

37 调对线组的保护比较到位，不容易切线，起鱼时更容易将杆立起来。适合新手或鱼情较为复杂的场景, 比较适合野钓。

28 调具有起鱼快速、控鱼有力的有时，可使用长竿短线、垂线钓法，适用于竞技赛中快速垂钓以及遛鱼，也适用于有水草以及障碍物的水域中使用，可以令鱼快速出水入护，适合钓大鱼时使用。但是这种调性的钓竿韧性较差，易发生断竿的现象，这也是其缺点所在。 比较适合有一定的钓鱼经验的人使用。

19 调材质好，碳纤维含量比28钓略高，竿体轻便、竿身挺直，钓大鱼起鱼快，但需有一定技巧，否则特别容易断竿，小点的鱼直接可以飞，所以很适合打比赛，休闲野钓不太适合。 只有钓鱼高手能够操控。

19 以上适合调性与 19 调接近， 但是由于杆非常硬，没有缓冲余地，因此这需要钓鱼大神来操控。

通过以上的对比， 本人选择 37 调鱼杆（我说呢，为什么最近老是跑鱼，我使用的是 19 调6.3m 的鱼杆）。　对于 19 钓鱼杆的话，可以加大线号，或可以直接采用大马力线作为主线或子线。
