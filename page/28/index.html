2013-07-25 实验记录

发表于 2013-07-25 更新于 2021-12-11 分类于 水合肼制备铜粉

通过前几次的实验，发现是很难摆脱出现铜镜现象的，团聚很严重的，一时找不到合适的分散剂，和反润湿剂， 所以已经放弃了有氢氧化铜合成铜， 通过前几次的实验还发现其实水合肼的需求量还是不算高， 我以为水合肼反应， 氮是被氧化为氮气的。 380毫升水溶解37克的葡萄糖， 加入120毫升的硫酸铜， 然后升温到80， 加入14.5克氢氧化钠和50毫升的水， 这时的温度为85， ph9， 待溶液成砖红色时， 加入20毫升的水合肼，155毫升水， 测ph 10, 但是保温很长时间都不完全反应， 所以我想氧化亚铜与水合肼反应的碱度要求比较高， 甚至在后来我把温度升到90基本没有很好的反应， 后来我把ph调到了13以后， 很快就反应了。 span class=”post_sig”published by wordpress for android /span

2013-07-24 实验记录

发表于 2013-07-24 更新于 2021-12-11 分类于 水合肼制备铜粉

实验一: 380毫升水, 2克pvp, 120毫升硫酸铜 ，10毫升硝酸银， 30毫升水合肼+145毫升水 380毫升水+120毫升硫酸铜， 溶解2克pvp, 加入10毫升硝酸银， 蓝色略带一点黄色， 升温到65度， 成墨绿色， 并且有明显的沉淀(发现恒压滴液漏斗有漏液到烧杯)， 升温到84， 滴入30毫升水合肼+145毫升水。保温在80-90度之间 2分钟后，温度80, 捎带黑色，有变清的趋势; 6分钟后， 温度80, 黑色加剧， 带一点黄色; 9分钟后，温度77， 黑色加剧， 杯壁未侵入溶液的捎带绿色沉淀，中间有少许黄色的泡沫; 13分钟后， 温度82, 现象与上面相同， ph3; 21分钟后，温度80， 黑色加剧，ph4; 25分钟后，温度83,  黑色捎带一点红色，ph5, 表面红棕色泡沫较多，ph5; 44:分钟后，温度70, 红棕色， ph z6.5 不8表面红棕色泡沫减少， ； 46分钟后,温度78, 深红棕色，表面有少许白色泡沫 ph6.5；60分钟后， 温度80， 红棕色， 有大量气泡， 杯壁上有有铜镜现象， 未侵入溶液的有黑色颗粒。 现象总结 ，团聚有点严重， 铜粒子， 颗粒紧密， 洗料时发现还得加入碳氨 !–more– span class=”post_sig”published by wordpress for android /span
